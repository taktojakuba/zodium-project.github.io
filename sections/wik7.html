Overview

zykron ships a complete KDE Plasma desktop, pre-configured and polished from first boot. Feature-rich and familiar — without pulling in unnecessary packages or overriding your choices later.

Wayland session by default. X11 fallback available but not recommended — KDE Plasma on Wayland is solid and well-supported.

Switching to zykron

bash $ sudo bootc switch ghcr.io/zodium-project/zykron:latest

What's included

  • KDE Plasma desktop — latest stable release
  • Pre-configured themes, panel layout, and system settings
  • Wayland session by default, X11 fallback available
  • All of zcore's base tooling and defaults
  • No unnecessary KDE apps — only the core desktop components
  • KDE Discover disabled — use Flatpak and zync for packages

Who is this for?

zykron is for users who want a full-featured, traditional desktop experience — familiar mouse-driven workflow, system tray, file manager, and all the configuration panels KDE offers. If you want something leaner, look at zynori.
