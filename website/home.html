tools

Five focused utilities.

All written in Rust. All designed to slot cleanly into a minimal workflow without getting in the way.

zbox packages

Rootless container manager and distrobox replacement, built for zodium. Written in Rust.

→
zfetch system info

Fast system info fetch with clean visuals, multiple themes, and a built-in TUI config editor.

→
zrun launcher

TUI shell-script launcher with fuzzy search, tagging, and full run history. No external deps.

→
zgpu gpu

GPU selection and PRIME offload launcher. Intel, AMD, NVIDIA — static binary, no runtime deps.

→
zync updates

One command to update everything: system image, Flatpak, containers, firmware. Rollback included.

→
zbox containers

Distrobox replacement built for zodium. Manage rootless containers with a clean, minimal interface.
