Built on the Atomic Foundation

Harnessing the power of Fedora's immutable architecture with curated enhancements.

🏗️

OCI-Based

Distributed as container images via GitHub Container Registry. Updates are atomic, predictable, and easily rolled back.

🔒

Immutable Core

The system root is read-only. Say goodbye to configuration drift and accidental system breakage during updates.

📦

Container-First

Native support for Distrobox and Toolbx. Keep your host system clean while developing in any environment you need.

Frequently Asked

Since Ferret-Project is an OCI-based Fedora remix, you can install it by starting with a standard Fedora Atomic (Silverblue/Kinoite) installation and "rebasinng" to our image using the rpm-ostree rebase command.
We recommend Flatpaks for desktop applications and Distrobox/Toolbx for CLI tools and development environments. This keeps your base system clean and stable.
Ferret Project is an independent community project. While we use Fedora as our upstream base and leverage their excellent Atomic technology, we are not officially affiliated with the Fedora Project or Red Hat.
